**a) Carga Eléctrica: **

**Q(Carga electrica)=(numero de electrones)/(6.25 x 〖10〗^18 (e^-/C))**

     2. ¿Cuál es la carga en coulombs (C) del núcleo de un átomo de cloro?
     
Numero de electrones del cloro: 29 electrones.

Q(cloro)=(29 e^-)/(6.25 x 〖10〗^18 (e^-/C))=4.94 x 〖10〗^(-18)  C

     4. ¿Cuántos electrones se requieren para producir 80 μC  (microcoulombs) de carga?
     
C= 80 μC x  (1 x 〖10〗^(-6) C)/(1 μC)=8 x〖 10〗^(-5)  C 

#e^-=Q(coulombs)x 6.25 x 〖10〗^18 (e^-/C)

#e^-=8 x〖 10〗^(-5)  C x 6.25 x 〖10〗^18 (e^-/C)= 5 x 〖10〗^14

     6. Se utilizan quinientos joules de energía para mover 100 C de carga por un resistor. ¿Cuál es el voltaje a través del resistor?
     
V(voltaje)=(W(energia en joules)(J))/(Q(C))

V(voltaje)=(500(J))/(100 (C))=5 V

     8. ¿Cuánta energía utiliza una batería de 12 V para mover 2.5 C por un circuito?
     
W(energia en joules)(J)=V(voltaje)  x Q(C) 

W(J)=12 V x 2.5 C=30 J

     10. Determine la corriente en cada uno de los siguientes casos: 
     
I(corriente)(A)=(Q (C))/(t (tiempo)(s))

(a) 75 C en 1 s : I(corriente)=(75 C)/(1 s)=75 A

(b) 10 C en 0.5 s:  I (corriente)=(10 C)/(0.5 s)=20 A

(c) 5 C en 2 s: I (corriente)=(5 C)/(2 s)=2.5 A

     12. ¿Cuánto tiempo requieren 10 C para fluir más allá de un punto si la corriente es de 5 A?
     
t (tiempo)(s)=Q (C)x I(corriente)(A)  

t (tiempo)(s)=10 C x 5 A=50 s

     14. 5.74 x 〖10〗^17electrones fluyen por un alambre en 250 ms. ¿Cuál es la corriente en amperes?
     
Q(Carga electrica)=(5.74 x 〖10〗^17 electrones)/(6.25 x 〖10〗^18 (e^-/C))=0.09184

t (tiempo)(s)=250 ms x  (1 x 〖10〗^(-3))/(1 ms)=0.25 (s)

I(corriente)(A)=(0.09184 (C))/(0.25 (s))=0.36736 (A)
